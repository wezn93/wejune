CSS에서 사용할 수 있는 background 속성은 다음과 같다

  1. background-color
  2. background-image
  3. background-repeat
  4. background-position
  5. background-attachmet

background-color 속성은 해당 HTML요소의 배경색을 설정

background-image 속성은 해당 HTML요소의 배경으로 나타날 배경 이미지를 설정

background-repeat 속성은 배경 이미지를 수평이나 수직 방향으로 반복되도록 설정

background-position 속성은 반복되지 않는 배경 이미지의 상대 위치를 설정

background-attachmet 속성은 배경 이미지를 해당 위치에 고정시키도록 설정
